VORTEX INVENTORY MANAGEMENT SYSTEM - API TEST HANDBOOK (WITH AUTHENTICATION)
=============================================================================
Base URL: http://localhost/api_php/api

1. REGISTER AN ACCOUNT (POST)
-----------------------------
Creates a new user. The password is encrypted and saved securely.
* Request Route:
  POST http://localhost/api_php/api/register.php

* Headers:
  Content-Type: application/json

* Request JSON Body:
{
  "username": "nikhil_admin",
  "password": "my_secure_password_123"
}


2. USER LOGIN / GET SESSION TOKEN (POST)
---------------------------------------
Verifies user credentials. On success, issues a unique cryptographically secure bearer token valid for 8 hours.
* Request Route:
  POST http://localhost/api_php/api/login.php

* Headers:
  Content-Type: application/json

* Request JSON Body:
{
  "username": "nikhil_admin",
  "password": "my_secure_password_123"
}

* Success Response:
{
  "status": "success",
  "message": "Logged in successfully.",
  "data": {
    "username": "nikhil_admin",
    "access_token": "a1b2c3d4e5f6...",
    "token_type": "Bearer",
    "expires_at": "2026-07-10 18:00:00"
  }
}


3. USER LOGOUT / INVALIDATE SESSION (POST)
------------------------------------------
Invalidates the active session token in the database. Requires Authorization header.
* Request Route:
  POST http://localhost/api_php/api/logout.php

* Headers:
  Authorization: Bearer <your_access_token_here>
  Content-Type: application/json


4. AUTHENTICATED INVENTORY OPERATIONS
-------------------------------------
All requests to items.php MUST include the "Authorization: Bearer <token>" header.

* Fetch Inventory List:
  GET http://localhost/api_php/api/items.php
  Headers -> Authorization: Bearer <your_access_token_here>

* Filter List (Search & Category):
  GET http://localhost/api_php/api/items.php?search=mouse&category=Electronics
  Headers -> Authorization: Bearer <your_access_token_here>

* Get Item Details:
  GET http://localhost/api_php/api/items.php?id=1
  Headers -> Authorization: Bearer <your_access_token_here>

* Create Product (POST):
  POST http://localhost/api_php/api/items.php
  Headers -> 
    Authorization: Bearer <your_access_token_here>
    Content-Type: application/json
  Body ->
  {
    "name": "Smart Watch Series X",
    "sku": "SW-SERX-08",
    "description": "Fitness tracking smartwatch.",
    "price": 199.99,
    "quantity": 80,
    "category": "Electronics"
  }

* Update Product (PUT):
  PUT http://localhost/api_php/api/items.php?id=1
  Headers -> 
    Authorization: Bearer <your_access_token_here>
    Content-Type: application/json
  Body ->
  {
    "price": 44.99,
    "quantity": 115
  }

* Delete Product (DELETE):
  DELETE http://localhost/api_php/api/items.php?id=1
  Headers -> Authorization: Bearer <your_access_token_here>
